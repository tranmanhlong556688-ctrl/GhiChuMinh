# Ghi Chú Minh

Ứng dụng Android ghi chú đơn giản bằng Java.

## Chức năng

- Thêm ghi chú mới
- Bấm vào ghi chú để sửa
- Nhấn giữ ghi chú để xóa
- Lưu dữ liệu trên điện thoại bằng SharedPreferences
- Build APK tự động bằng GitHub Actions

## Lấy APK

Vào tab **Actions** trên GitHub, mở workflow **Build APK**, tải artifact **GhiChuMinh-debug-apk**.
